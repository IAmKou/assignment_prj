## DESKTOP AIMS APP WITH JAVAFX
### Hưỡng dẫn chạy app
Requirement:
 - Docker desktop
 - Maven
 - Java 21
```bash
  # to create db
  docker compose up
  # create to test and package project to .jar
  mvn install 
  java -jar ./target/aims-0.0.1-SNAPSHOT.jar
```
